# Santoshidevi - Yoga & Wellness

A modern, responsive single-page application (SPA) for Santoshidevi yoga brand. This platform showcases yoga courses, products, online classes, and wellness services.

## Features

- Multi-language support (English/Spanish)
- Responsive design
- Product/Service showcase
- Contact forms for inquiries
- CTA buttons for course enrollment and product purchases

## Tech Stack

- React 18
- Vite
- Styled Components
- i18next for internationalization
- React Icons

## Getting Started

```bash
npm install
npm run dev
```
